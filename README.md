# 🏃 RunClub — Frontend

เว็บแอปพลิเคชันสำหรับชมรมวิ่ง ช่วยให้สมาชิกสามารถบันทึกผลการวิ่ง ดูสถิติส่วนตัว และติดตามความก้าวหน้าของตัวเองได้

---

## ✨ ฟีเจอร์หลัก

- 🔐 **ระบบ Auth** — สมัครสมาชิก / เข้าสู่ระบบ พร้อม JWT Token
- 👤 **ตั้งค่าโปรไฟล์** — บันทึกข้อมูลส่วนตัว (น้ำหนัก, ส่วนสูง, อายุ)
- 📊 **Dashboard** — แสดงสถิติการวิ่ง สรุประยะทาง ความเร็ว และแคลอรี่
- 📝 **บันทึกการวิ่ง** — เพิ่มบันทึกการวิ่งแต่ละครั้งผ่าน Modal
- 🔒 **Protected Routes** — หน้าที่ต้องล็อกอินก่อนถึงจะเข้าได้

---

## 🛠 Tech Stack

| ด้าน | เทคโนโลยี |
|------|-----------|
| UI Framework | React 19 + TypeScript |
| Routing | React Router v7 |
| Styling | Tailwind CSS v4 |
| Build Tool | Vite 8 |
| API | REST API (Backend แยกโปรเจกต์) |
